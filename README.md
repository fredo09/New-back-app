### Instalación 🔧

_Una serie de ejemplos paso a paso que te dice lo que debes ejecutar para tener un entorno de desarrollo ejecutandose_

_Dí cómo será ese paso_

1. Docker

Para poder ejecutar con comado de docker usar la siguiente instruccion.

```
  docker-compose up -d
```

Para bajar el contenedor.

```
  docker-compose down
```

2. Para correro independientemente de docker usar lo siguiente:


```
  npm run start:dev
```
